0. Chọn loại dữ liệu đang tải lên

Chọn "Dữ liệu cả tháng" nếu 3 file bên dưới là báo cáo tổng hợp cho cả tháng (sẽ hiển thị ở lựa chọn "Dữ liệu tháng"). Chọn "Dữ liệu 1 ngày cụ thể" và chọn ngày nếu 3 file bên dưới chỉ là báo cáo của riêng 1 ngày (sẽ hiển thị như một mục riêng theo ngày đó). Lựa chọn này áp dụng chung cho cả 3 file Campaign / Ads / Lead bên dưới — nhớ kiểm tra lại trước khi bấm các nút cập nhật.

Đang tải lên cho: Dữ liệu cả tháng.

1. File dữ liệu Campaign

Chấp nhận file Excel (.xlsx/.xls) hoặc CSV. Sau khi chọn file, chọn cột tương ứng bên dưới rồi bấm "Cập nhật Campaign". Giá trị trạng thái "active" hiển thị là Đang chạy, "not_delivering" hiển thị là Tạm dừng. Dữ liệu này sẽ thay thế toàn bộ bảng ở tab "Theo Campaign".

2. File dữ liệu Ads ID

Chấp nhận file Excel (.xlsx/.xls) hoặc CSV. Sau khi chọn file, chọn cột tương ứng bên dưới rồi bấm "Cập nhật Ads". Giá trị trạng thái "active" hiển thị là Đang chạy, "not_delivering"/"inactive" hiển thị là Tạm dừng. Nếu file có cột số lượng tin nhắn (messaging conversations), chọn cột đó để hệ thống tự tính tỉ lệ chuyển đổi CR = Số lead / Số tin nhắn ở tab "Theo Ads". Nếu file có cột Campaign ID, chọn cột đó để hệ thống tự cộng dồn Số lead / Số tin nhắn theo từng campaign và hiển thị ở tab "Theo Campaign".

3. File Lead theo Ads ID

Chấp nhận file Excel (.xlsx/.xls) hoặc CSV. Sau khi chọn file, chọn cột tương ứng bên dưới rồi bấm "Cập nhật Lead". File này sẽ tự động kết nối với file Ads ID (mục 2) theo Ads ID để tính cột "Số lead" và "Chi phí Lead" ở tab "Theo Ads".

Chọn file, kiểm tra/chỉnh lại cột cho khớp, bấm nút cập nhật tương ứng, sau đó chuyển sang tab "Theo Campaign" hoặc "Theo Ads" để xem dữ liệu mới. Có thể cập nhật lại bất kỳ lúc nào bằng cách tải file mới.
